Auth Multi-Tenant

Login por empresa. Cada usuário pertence a uma empresa e vê apenas os dados dela. Roles: superadmin, admin, gestor, engenheiro, financeiro, viewer.

✓ Implementado
📊

Dashboard

Visão geral com cards de obras ativas, saldo gerencial, provisões, gráfico de fluxo de caixa (6 meses) e obras por status. Recharts integrado.

✓ Implementado
👥

Clientes

Cadastro PF/PJ completo com CPF/CNPJ, endereço, contatos. Busca por trigram (pg_trgm). Modal de criação/edição.

✓ Implementado
📦

Banco de Insumos

Base própria + bases públicas (SINAPI, FDE, ORSE, DEINFRA, EMOP). Filtro por fonte, categoria, código. Histórico de preços por mês de referência.

✓ Implementado
🧩

Composições

Composições sempre baseadas em insumos. Trigger automático recalcula custo total ao adicionar/editar itens. Expandir linhas para ver ingredientes.

✓ Implementado
📄

Orçamentos

WBS por grupos/etapas. Itens por composição ou insumo direto. BDI por item ou global. Trigger recalcula valor_custo, valor_bdi e valor_total.

✓ Implementado
🏦

Visão Geral Financeira

Cards dos 5 tipos de conta, gráfico de barras horizontal com saldos por tipo. Últimos lançamentos do mês com receitas/despesas/provisões.

✓ Implementado
💳

Contas Financeiras

Criação das 5 contas seguindo o fluxograma: Gerencial, Operacional (por obra), Fluxo Fixo, Provisões (subtipo) e Lucro Excedente.

✓ Implementado
↔️

Lançamentos

Receitas, despesas, provisões e transferências. Filtros por mês, tipo, status. Confirmar pagamento atualiza saldo da conta via trigger SQL.

✓ Implementado
⚡

Provisões

4 tipos: trabalhistas (INSS, FGTS, rescisões), tributárias (PIS, COFINS, CSLL, IRPJ, ISS), marketing (3,5%), lucro. Regras automáticas por percentual.

✓ Implementado
📐

Medições de Obra

Medições periódicas vinculadas ao orçamento. Controle de % acumulado, valor medido e aprovação. Status: rascunho → enviada → aprovada → paga.

⚙ Em breve
🏗

Gestão de Obras

7 status de ciclo de vida: prospecção → orçamento → contratada → em andamento → paralisada → concluída → cancelada. Visualização cards ou tabela.

✓ Implementado
📓

Diário de Obras

Registro diário com data, clima, nº de funcionários, % de execução e fotos. Histórico completo de ocorrências da obra.

✓ Schema · UI em breve
📎

Documentos

Anexos por obra: contratos, plantas, ARO, NF, fotos. Upload via Supabase Storage (bucket: obras-documentos). Signed URLs para acesso seguro.

✓ Schema · UI em breve
👷

Equipe da Obra

Alocação de usuários do sistema ou colaboradores externos. Controle de entrada/saída por período em cada obra.

✓ Schema · UI em breve
📃

Contratos

Contratos vinculados a obra + orçamento + cliente. Tipos: obra, serviço, fornecimento, manutenção. Histórico de status e arquivo PDF.

✓ Schema · UI em breve
📊

Relatórios

DRE por obra, fluxo de caixa mensal, comparativo orçado vs realizado, curva S de execução, rentabilidade por obra.

⚙ Em desenvolvimento
📬

Fornecedores

CRUD completo PF/PJ. Categorias: materiais, serviços, equipamentos. Vinculação a lançamentos de despesas.

⚙ Em desenvolvimento
⚙

Configurações

Dados da empresa, logo, plano, regras de provisão padrão, usuários e permissões, integração com bases SINAPI/FDE.

⚙ Em desenvolvimento
🏢

Multi-Empresa (SaaS)

Cadastro público de nova empresa, planos (free/starter/pro/enterprise), billing, isolamento total de dados por tenant.

⚙ Em desenvolvimento
📱

PWA / Mobile

Versão mobile para engenheiros em campo: diário de obras, fotos, % execução, lançamentos rápidos.
